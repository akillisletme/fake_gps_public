# Features

## 4 Simulation Modes

| Mode | Description |
|------|-------------|
| **Basic Mock** | Tap anywhere on the map to instantly set your location |
| **Joystick** | Move your location in real time with an on-screen joystick |
| **Route Simulation** | Place waypoints and simulate traveling along a route |
| **Human Simulation** | Natural movement within a radius (walking, cycling, driving) |

---

## Route Tools

- Add, drag, and reorder waypoints on the map
- Tap near a segment to insert a new waypoint
- Undo last waypoint
- Save and reload routes from the drawer
- Set route duration (arrive in X minutes)

---

## Scheduler

- Schedule a location to activate at a specific time
- One-time or recurring (daily, selected weekdays)
- Works via Android AlarmManager — triggers even when the app is closed
- Automatically rescheduled after device reboot

---

## Favorites

- Save frequently used locations with custom names
- Quick access from the side drawer
